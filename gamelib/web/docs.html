General

Antes de nada, para usar la librería hay que incluir la cabecera gamelib/gamelib.h y a la hora de enlazar hay que incluir las librerías: -lgamelib -lopengl32 -lSDLmain -lSDL -lSDL_image -lSDL_mixer -lSDL_ttf -lSDL_net

GameLib aranca automáticamente sin hacer nada mas que incluir la cabecera. Aún así, lo primero que habrá que hacer es crear una ventana con el modo gráfico deseado con VideoModeSet.

A partir de ese momento, para cada frame, deberás llamar a VideoModeEnable2D y después de pintar por pantalla todo lo que quieras, VideoModeDisable2D y a VideoModeFrame para actualizar el teclado, mantener la aplicación viva y esperar el tiempo necesario para que el juego vaya a una velocidad constante de 60 FPS (por defecto).

Adicionalmente se pueden utilizar todas las funciones de SDL, SDL_Image, SDL_Mixer, SDL_net, SDL_ttf y OpenGL para conseguir resultados mas potentes.

Modo Gráfico

Funciones Básicas
void VideoModeSetTitle(char *title);

Define el título de la ventana especificado por el parámetro title

void VideoModeSet(int width, int height, int windowed);

Crea una ventana del tamaño especificado por width y height. El parámetro window establece si el juego será en ventana o a pantalla completa. Un número distinto de 0 indicará en ventana, y el valor 0 a pantalla completa.

void VideoModeEnable2D();

Prepara OpenGL para renderizar objetos en 2 dimensiones.

void VideoModeDisable2D();

Termina de renderizar objetos en 2 dimiensiones.

void VideoModeFrame();

Función que se encarga de actualizar el teclado, mostrar por pantalla los cambios realizados, esperar el tiempo necesario para mantener una velocidad constante en el juego y borrar el buffer.

void ColorSet(float r, float g, float b, float a);

Especifica un color RGBA (rojo, verde, azul y alpha) que se usará para dibujar primitivas y texto. Las componentes son floats cuyo valor debe estar en el rango [0.0f, 1.0f]

Variables Globales
extern int screenWidth, screenHeight;

Indican el ancho y el alto de la ventana respectivamente una vez creada. Para hacer referencia al centro de la pantalla usaríamos (screenWidth / 2) y (screenHeight / 2)

Funciones Extendidas
void VideoModeSetEx(int widthScreen, int heightScreen, int widthDraw, int heightDraw, int windowed);

Función similar a VideoModeSet, a excepción de que permite definir un tamaño virtual. La ventana puede ser, por ejemplo de 800x600 pero trabajar a 640x480, que se reescalará. Esto permite trabajar con la comodidad de una única resolución, pero poder tener varias.

void VideoModeSetFPS(int fps);

Se encarga de cambiar los frames por segundo del juego (la cantidad de fotogramas que se verán por cada segundo).

int VideoModeGetFPS();

Obtiene los FPS actuales. Por defecto son 60.

Teclado

Variables Globales

Las variables globales key y keyv se actualizan al llamar a las funciones VideoModeFrame y/o KeyboardUpdate. Por regla general para la entrada de teclado, se usarán siempre estas variables globales y rara vez las funciones extendidas.

extern int keyv[SDLK_LAST];

keyv es un vector que contiene un valor distinto de 0 si una tecla determinada está en estos momentos pulsada. Ésta se usa para movimientos continuos, sin embargo para introducir texto o elegir entre opciones de menús key debería ser la mejor opción.

extern char key[SDLK_LAST];

key es un vector que contiene un valor distinto de 0 si la tecla se ha pulsado. Esta variable tiene un retraso de repetición con acleración y puede ser útil para entrada de texto y movimiento en menús con posiciones discretizadas.

Constantes

Contienen los índices de los vectores key y keyv para esas teclas:

_a, _b, _c, _d, _e, _f, _g, _h, _i, _j, _k, _l, _m, _n, _o, _p, _q, _r, _s, _t, _u, _v, _w, _x, _y, _z

_up, _down, _left, _right

_esc, _enter, _space, _lshift, _rshift

Ejemplo: while (!key[_esc]) { ... } ejecutaría un código mientras no se pulse la tecla escape.

Funciones Extendidas
void KeyboardUpdate();

Actualiza las variables key y keyv. No se suele usar ya que la función VideoModeFrame ya la llama internamente.

void KeyboardSetDelay(int delay, int interval);

Cambia el valor de del retraso entre repeticiones del teclado.

Imágenes

Estructuras 
// Declaramos el tipo "Image"
typedef struct _Image _Image;
typedef struct _Image *Image;

// Estructura de la Imagen
struct _Image {
    // Imagen padre para subimágenes
    Image father;

    // Coordenadas de la subimágen en la imagen real
    int x, y, w, h;

    // Centro de la imagen
    int cx, cy;

    // Ancho y altos reales de la imagen
    int rw, rh;

    // Textura OpenGL
    GLuint gltex;

    // CallList para renderización rápida
    GLuint callList;

    // Lista de puntos en la imágen
    struct { float x, y; } texp[2];
};
Funciones de Carga y Descarga

GameLib utiliza SDL_Image para la carga de imágenes. Actualmente SDL_Image soporta los siguientes formatos: BMP, GIF, JPEG, LBM, PCX, PNG, PNM, TGA, TIFF, XCF, XPM, XV

Image ImageCreate(int w, int h);

Crea una imagen vacía de tamaño wxh

Image ImageCreateFromSubImage(Image vpi, int x, int y, int w, int h);

Crea una imagen a partir de un recorte de otra mas grande. La textura no se duplica, así que en la memória gráfica ocupa lo mismo.

Image ImageLoadFromStream(Stream s);

Carga una imagen a partir de un stream.

Image ImageLoadFromMemory(void *ptr, int length);

Carga una imagen a partir de un bloque de memória determinado por un puntero y una longitud.

Image ImageLoadFromFile(char *filename);

Carga una imagen a partir de un fichero.

Image ImageDuplicate(Image i);

Duplica una imagen. La imagen tiene distintos atributos, pero en la memória de video, representa la misma textura. Sirve para poder definir diferentes imágenes con diferentes estados, por ejemplo diferente centro de imagen.

void ImageFree(Image i);

Libera una imagen creada por cualquiera de las funciones de arriba.

Funciones de Información
void ImageSetCXY(Image i, int cx, int cy);

Establece el punto de centrado de una imagen. Al cargar o crear una imagen, el punto de centrado por defecto es el 0, 0. Una operación muy común es establecer como centro, el propio centro de la imagen: ImageSetCXY(image, image->w / 2, image->h / 2);

Funciones de Renderizado
void DrawImage(Image i, int x, int y);

Permite dibujar una imagen sobre la pantalla en las coordenadas especificadas.

void DrawImageEx(Image i, int x, int y, float size, float angle, float alpha);

Permite dibujar una imagen sobre la pantalla en las coordenadas especificadas con atributos de tamaño, ángulo y alpha. Un tamaño de 1.0 es el tamaño de la imagen original, 2.0 el doble y 0.5 la mitad. Los ángulos se miden en grados [0.0f, 320.0f]. Y el alpha es un valor entre [0.0f, 1.0f]; 1.0f es una imagen opaca, un vakir 0.0f es una imagen completamente transparente (no se ve)

Funciones de Renderizado sobre Texturas
int ImageStartDrawing(Image i);

Función que al llamarse, permite dibujar sobre una imagen. Devuelve 0 si tuvo éxito. Una vez llamada con éxito, hará que todas las funciones de dibujado, afecten a la imagen en vez de a la pantalla hasta que se llame a ImageStopDrawing.

void ImageStopDrawing();

Esta función se llama cuando se ha terminado de dibujar sobre la textura, para volver a dibujar sobre la pantalla.

Funciones de Renderizado de primitivas

Estas funciones permiten dibujar primitivas sobre la pantalla con un color especificado por la función ColorSet

void DrawLine(int x1, int y1, int x2, int y2);

Dibuja una linea que une los dos puntos especificados.

void DrawRectangle(int x1, int y1, int x2, int y2);

Dibuja un rectángulo determinado por los dos puntos especificados.

void DrawFilledRectangle(int x1, int y1, int x2, int y2);

Dibuja un rectángulo relleno determinado por los dos puntos especificados.

void DrawClear();

Borra toda la pantalla (o imagen).

Fuentes

Estructuras 
typedef struct {
    int    h;
    int    w[0x100];
    GLuint textures[0x100];
    GLuint listBase;
} *Font, _Font;
Funciones de Carga y Descarga
Font FontLoadFromFile(char *filename, int height);

Carga una fuente TTF a partir de un fichero. Hay que especificarle el alto de dicha fuente.

Font FontLoadFromMemory(void *ptr, int length, int height);

Carga una fuente TTF a partir de un bloque de memória, determinado por un puntero y una longitud. Hay que especificarle el alto de dicha fuente.

Font FontLoadFromStream(Stream s, int height);

Carga una fuente TTF a partir de una stream. Hay que especificarle el alto de dicha fuente.

void FontFree(Font f);

Libera una fuente cargada con algunas de las funciones FontLoad.

Funciones de Información
int FontWidth(Font f, char *text);

Obtiene el ancho que ocuparía un determinado texto con una fuente determinada. Adicionalmente se puede obtener el alto de la fuente utilizando el campo h de la estructura Font: font->h

Funciones de Dibujado
void DrawFontText(Font font, int x, int y, char *text);

Dibuja un texto en las coordenadas x, y con la fuente especificada.

void DrawFontTextf(Font font, int x, int y, char *format, ...);

Dibuja un texto con formato en las coordenadas x, y con la fuente especificada. El formato es el mismo utilizado en las funciones (s)printf

Sonido

Variables Globales
extern Channel freeChanel;

Es el canal -1 que siempre será un canal libre

Funciones de Música

La música de fondo tiene un canal propio. Se puede reproducir como música ficheros MP3, OGG, MOD, IT, XM, S3M. Los archivos de música solo se peuden cargar desde ficheros.

Music MusicLoadFromFile(char *filename);

void MusicFree(Music m);

Libera una música cargada con MusicLoadFromFile.

int MusicPlay(Music m);

Reproduce una canción especificada.

int MusicPlayEx(Music m, int loops, int fadems, int position);

Reproduce una canción en un canal especificado, un número determinado de veces (loops = 0 --> 1 vez), la cantidad de milisegundos para hacer un fadein y la posición desde dónde comienza.

void MusicFadeOut(int ms);

Crea un fadeout de la música con una duración en milisegundos. Al terminar, el sonido será pausado.

void MusicStop();

Detiene la reproducción de la canción.

void MusicSetVolume(float v);

Especifica el volumen de reproducción de la canción.

float MusicGetVolume();

Obtiene el volumen de reproducción de la canción.

Funciones de Muestras y Canales

Los sonidos pueden ser ficheros WAV, MP3, OGG, se pueden cargar de Streams, Memória y Ficheros. Las muestras de sonido se reproducen en canales. Solo se puede reproducir u nsonido a la vez por canal. Si hay varios sonidos reproduciéndose a la vez en varios canales, se mezclarán.

Sound SoundLoadFromStream(Stream s);

Carga un sonido desde un Stream.

Sound SoundLoadFromMemory(void *ptr, int length);

Carga un sonido desde una zona de memoria.

Sound SoundLoadFromFile(char *filename);

Carga un sonido desde un fichero.

void SoundFree(Sound s);

Libera un sonido cargado con las funciones de arriba.

Channel SoundPlay(Sound s);

Reproduce un sonido en un canal libre y devuelve dicho canal.

Channel SoundPlayEx(Sound s, Channel channel, int loops, int fadems, int ticks);

Reproduce un sonido en un canal especificado, un número determinado de veces (loops = 0 --> 1 vez), la cantidad de milisegundos para hacer un fadein y los ticks desde dónde comienza.

void ChannelPause(Channel c);

Pausa la reproducción de un sonido en el canal especificado.

void ChannelResume(Channel c);

Reanuda la reproducción de un sonido en el canal especificado.

void ChannelStop(Channel c);

Detiene la reproducción de un sonido en el canal especificado.

void ChannelStopAfter(Channel c, int ms);

Detiene la reproducción de un sonido en el canal especificado pasados una cantidad de milisegundos.

void ChannelFadeOut(Channel c, int ms);

Crea un fadeout del sonido en el canal especificado con una duración en milisegundos. Al terminar, el sonido será pausado.

bool ChannelIsPlaying(Channel c);

Indica si en el canal especificado se está reproduciendo un sonido..

Números Aleatorios

Funciones de Generación de Números Aleatorios
void genrand_init();

Inicializa el módulo de generación de números aleatorios. Se usan los segundos y milisegundos del sistema para determinar la semilla.

int genrand_between(int from, int to);

Obtiene un número al azar entre los valores from y to especificados.
